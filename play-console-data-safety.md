## 2. What type of data is collected?

### Financial info
- **Purchase history**: Subscription purchases (Weekly Pro, Monthly Pro, Super Pro)
- **Payment method**: Processed by Razorpay/Stripe — we do NOT store card/UPI details

### Personal info
- **Name**: Display name from Google Sign-In
- **Email address**: Google Sign-In email (used as user ID)
- **Photos and videos**: Homework images uploaded for AI solving

### Device or other IDs
- **Device ID**: Generated unique identifier stored locally for quota tracking
- **Advertising ID**: Not collected
- **Firebase Installation ID**: Auto-generated by Firebase SDK for analytics/crash reporting

## 3. How is this data used?

| Data type | Purpose |
|-----------|---------|
| Email, name | Account creation, authentication, subscription management |
| Homework images | AI-powered problem transcription and solving (sent to Google Gemini) |
| Payment info | Processing subscription purchases (via Razorpay/Stripe) |
| Device ID | Quota tracking, free trial abuse prevention |
| Crash logs | App stability monitoring (Firebase Crashlytics) |
| Analytics | Understanding usage patterns to improve the app (Firebase Analytics) |

**Answer: App functionality, Analytics, Fraud prevention, Security**

## 4. Is this data required or can the user choose not to share it?

| Data type | Required? | Can opt out? |
|-----------|----------|-------------|
| Email, name | Required for Google Sign-In | No (can use Guest mode instead) |
| Homework images | Required for core functionality | No (essential to use the app) |
| Payment info | Required for paid plans | No (only needed for subscriptions) |
| Device ID | Required for quota tracking | No (used for service functionality) |
| Crash logs | Not required | Yes (can disable in Profile Settings) |
| Analytics | Not required | Yes (can disable in Profile Settings) |

**Answer: Some data is required, some is optional**

---

## 5. Is this data encrypted in transit?

**Answer: Yes**

- All data transmitted over HTTPS/TLS encryption
- Backend hosted on Render with SSL
- Payment data encrypted by Razorpay/Stripe (PCI DSS compliant)

---

## 6. Can users request deletion of their data?

**Answer: Yes**

- Account deletion available in Profile Settings
- Deletes: email, homework images, solve history, feedback, subscription status
- Retains: `was_ever_paid` and `ever_used_free_trial` flags (for abuse prevention)
- Payment records retained 8 years (Indian tax law requirement)
- Users can contact Grievance Officer for additional requests

**Grievance Officer**: Girish Shankaran — snapsolve.edupp@gmail.com (7-day response)
